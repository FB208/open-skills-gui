# Open Skills GUI v1 执行计划

## 目标

交付可在 Windows 10 22H2 / Windows 11 x64 上安装、运行、管理全局 Skill 并通过 GitHub Release 自动升级的 Open Skills GUI v1。

## 阶段

- [x] 阶段 1：建立工程、需求、版本和测试基线
- [x] 阶段 2：实现共享类型、状态存储、运行环境与命令适配层
- [ ] 阶段 3：实现扫描、接管、搜索、安装、启停、卸载和备注
- [ ] 阶段 4：实现 Skill 更新检查、更新冲突保护和软件更新
- [ ] 阶段 5：实现简体中文 Preact 界面及 Neutralinojs 通信
- [x] 阶段 6：实现 Windows 安装器、GitHub Actions 和发布校验
- [ ] 阶段 7：运行测试、构建、安装包验证并修复问题（进行中）
- [x] 阶段 8：重构 PC 客户端界面、简化交互并完成视觉验收

## 固定决策

- 仅支持 Windows x64 和全局 Skill。
- 使用 Neutralinojs、Preact、TypeScript、Node 扩展和 Inno Setup。
- 固定 `skills@1.5.19`；禁用其遥测，不使用其会修改文件的 `check` 作为只读检查。
- 卸载软件保留用户数据、Skill 和目录联接。
- 不提供 Skill 介绍、遥测、便携包、代码签名或静默强制升级。
- 不做旧数据迁移；状态格式固定为 `schemaVersion: 1`。
- 前端只适配正常 PC 显示器，不新增移动端或窄屏响应式方案。
- 应用外壳占满初始窗口，页面根节点不产生默认滚动条；仅内容区域按需内部滚动。
- 视觉与交互采用轻量桌面客户端范式，减少网页式大标题、卡片堆叠和冗余说明。

## 验收条件

- 类型检查、单元测试、Windows 集成测试和生产构建通过。
- GitHub Actions 能产出唯一安装包 `OpenSkillsGUI-Setup-x64.exe`。
- 关键管理流程具备错误处理、路径保护、本地修改保护和中文界面。
