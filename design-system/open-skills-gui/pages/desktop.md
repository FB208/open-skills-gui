# Open Skills GUI：桌面客户端覆盖规范

> 本文件覆盖 `../MASTER.md` 中与桌面管理工具不相符的营销页、夸张标题、暗色主题和响应式建议。

## 产品边界

- 仅适配 Windows PC；不设计手机、平板或窄屏导航重排。
- 兼容正常 PC 显示器及 Windows 125%–150% 显示缩放。
- 业务优先级：浏览已安装 Skill > 搜索与筛选 > 更新/启停 > 低频设置。

## 布局

- 应用根节点固定为窗口高度，`html`、`body`、`#app` 和应用外壳均禁止页面级滚动。
- 使用 172px 固定纵向侧栏与 `minmax(0, 1fr)` 工作区；任何宽度下都不切换为顶部导航。
- 工作区采用三段结构：58px 页头、50px 工具栏、剩余内容区。
- 只有列表、搜索结果和设置内容区允许内部纵向滚动；禁止横向滚动。
- 不使用居中的网页最大宽度容器，工作区应自然占满客户端窗口。

## 信息层级

- 页面标题 18px/600，说明与数量摘要 12px；移除英文眉题和营销式大标题。
- 四张统计卡合并为单行文字摘要，例如“3 个 Skill · 3 已启用 · 0 可更新”。
- Skill 使用紧凑列表行，不使用独立悬浮卡片；行高约 72px，名称、状态、来源和目标两层展示。
- 每行只常驻主操作：有更新时显示“更新”，以及“启用/禁用”；检查、备注、卸载放入“更多”菜单。
- 搜索结果使用单列列表；设置使用分组面板，避免卡片套卡片。

- 字体使用 Windows 系统字体：Segoe UI Variable、Segoe UI、Microsoft YaHei UI。
- 间距只使用 4 / 8 / 12 / 16 / 24px；圆角以 6 / 8 / 10px 为主。
- 阴影仅用于菜单、对话框和浮动反馈；普通面板与列表使用边框分层。
- 图标继续使用项目内统一 SVG 线性图标，不引入图标字体或 Emoji。

## 交互

- 颜色、边框、背景过渡 120–180ms；禁止卡片上浮、缩放和装饰性滚动动画。
- 点击目标至少 32px；图标按钮必须提供可访问名称与 tooltip。
- 键盘焦点始终可见；原生对话框继续承担焦点约束和 Escape 行为。
- 支持 `prefers-reduced-motion`，忙碌态与操作结果继续通过进度条、Toast 和 ARIA 实时区反馈。

## 明确禁止

- 禁止页面根滚动条、移动端媒体查询、顶部移动导航和横向滚动。
- 禁止大面积径向渐变、玻璃拟态、拟物阴影、超大数字卡与卡片 hover 位移。
- 禁止依赖 hover 才能访问关键操作；更多菜单必须可键盘访问。
- 禁止添加 Tailwind、组件库、在线字体或新的运行时视觉依赖。

## 验收

- 默认窗口打开后无页面级横向或纵向滚动条。
- 真实启动首屏能看到完整页头、工具栏和至少三条 Skill 列表行。
- Windows 显示缩放不会触发另一套导航布局。
- 960×620 最小窗口仍保持纵向侧栏，所有主要操作可访问。
- 类型检查、UI 单元测试和正式构建通过；减少动画规则保留。
